# 娯疾駆

作詞：和鳳
-----

```
愛に似ているその羽
動かして
彷徨う私の灯火

黒の映えるに貴方は
見初められ
白い手を握る優しく

追い掛けた夜空の音
息溶かす歌声に

手を伸ばしたまま笑う
揺れるままに遊ぶように
風を受けて舞う黒の羽根
貴方からの幸せ胸に
娯楽唄


青に散るのは歪な
藍の音
結んだら開く哀色

指絡む吐息は鳴き
月明かり朧気に

踊るかのように謳う
弧成す永久に指を通す
響く笑い声照らしゆく
白と白が重なり光る
疾駆唄


透けるのは朱色の意図
香るまま久遠の華

徒に微笑むまま
娯疾駆に口付ける

黒白であるのならば
彩で殺め
あやす痩躯
光の飛沫は今も舞い
ずっと二人寄り添う為の
黒の唄
```
