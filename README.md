# な～にP

作詞：和鳳（なげは）と作曲：niu（にぅ / にュウ）によるユニット。二人の頭文字とボカロPのような感じを合わせて「な～にP」。

2009年頃にPiaproのユーザーコラボで出会い、数作の楽曲を残す。

2010年にミニアルバム「疾鵠」をリリース。その中の一曲である「迷響詩酔」はJOYSOUNDで配信中。
